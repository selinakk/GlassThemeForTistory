# 티스토리 블로그 글래스 테마 스킨
Glassmorphism UI를 적용한 티스토리 블로그 반응형 '글래스 테마' 스킨입니다<br/>
iOS 26의 liquid glass UI를 참고해 만들었습니다

<img src="https://github.com/selinakk/GlassThemeForTistory/blob/main/preview560.jpg?raw=true" alt="글래스 테마 스킨 프리뷰">


---


### 주요 특징
- 블로그 광고, SEO(검색 엔진 최적화)를 고려하여 만들었습니다
- 티스토리의 Odyssey 스킨에서 리뉴얼 하였습니다
- 사이드바 SNS 플러그인(Facebook, Twitter)은 제거했습니다
- 카테고리의 🔥이모지는 새 글 표시입니다
- 제 블로그를 기준으로 제작된 스킨이라 누락된 기능이나 부분이 있을 수 있습니다


### 활용 
- 본 테마는 자유롭게 수정 가능합니다
- style.css에서 :root의 --primary-color 값을 바꾸면 일부 포인트 색상을 커스텀 가능합니다
- PC 레이아웃 '2단 우측 사이드바'를 기준으로 제작하였으니 해당 레이아웃 사용을 권장합니다
- 글 목록 타입에서 '**이미지 크기 고정 타입**'은 썸네일이 모두 동일한 비율로 출력되며 열은 기본 3단, 해상도 가로 760px 이하에서는 2단입니다
- 글 목록 타입에서 '**이미지 비율 유지 타입**'은 썸네일이 이미지의 기존 비율대로 출력됩니다 따라서 각 썸네일 이미지 비율이 각양각색인 경우 행을 맞추기 위해 텍스트 하단에 여백이 생깁니다<br/>열은 기본 2단, 해상도 가로 576px 이하에서는 1단입니다
- 글 목록 타입에서 '**포스터 타입**'은 썸네일이 세로 포스터 비율이며 목록 가로 스크롤이 가능합니다
- 글 목록 타입에서 '**단순 텍스트 타입**'은 썸네일 이미지 없이 제목 및 본문 3줄을 출력합니다
- 문제 발견 시 블로그에 댓글 남겨주시면 확인 후 업데이트하겠습니다
